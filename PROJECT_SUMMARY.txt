================================================================================
CONFIGURATION OPTIONS
================================================================================

MIN_IA_PROBABILITY (default: 0.7)
  - Minimum supernova classification probability
  - Lower = more candidates, includes marginal cases
  - Higher = fewer candidates, higher confidence

DAYS_BACK (default: 30)
  - Query window in days
  - Broader window = includes older candidates
  - Narrower = recent discoveries only

USE_CACHE (default: True)
  - Use previously cached data
  - First run queries live data (slower)
  - Subsequent runs use cache (faster)
